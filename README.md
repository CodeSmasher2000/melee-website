# Three.js Ocean Website

This project is a standalone website based on the Three.js ocean example (`webgl_shaders_ocean.html`) adapted to run from one local file plus CDN imports.

## Prerequisites

- Node.js 18+

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Vite will print a local URL, usually `http://localhost:5173`.

## Production Build

```bash
npm run build
npm run preview
```

## Project Structure

- `index.html`: Complete website and ocean scene code.
